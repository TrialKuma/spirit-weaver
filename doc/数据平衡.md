## 伤害公式

```
最终伤害 = max(1, baseDamage × modifiers — enemyDef)
baseDamage = buffedAtk × baseMultiplier
buffedAtk = baseAtk × (1 + buff%) × (1 + 职业被动%)
```

## 速度与行动频率

- AV = 10000 / Speed
- Speed 100 ≈ 200 tick 行动一次
- Speed 200 ≈ 100 tick（频率翻倍）
- Speed 300（基础）≈ 33 tick 行动一次
- 剑圣满疾风(+100速) → Speed 400

---

## 强化数值规范

- R 级基础值：倍率类 +25%，属性类 +2~30
- SR = R × 2，SSR = R × 4
- 叠加公式：`value = baseValue × (1 + 0.2 × (level-1))`

---

## 敌人数值锚点

- 基础范围：HP 100~300, ATK 9~18, DEF 0~5, Speed 55~110
- 精英：HP 280~300, ATK 14~16, DEF 3~4, Speed 85~120
- Boss：HP 500, ATK 18, DEF 6, Speed 75
- 每阶段缩放：HP +20%, ATK +10%
- 精英额外：HP ×1.5, ATK ×1.3
- Boss 额外：HP ×2.0, ATK ×1.5

---

## 修改原则

1. 新技能伤害倍率参考同职业同类型技能，不要凭空设数
2. 新敌人 HP/ATK 参考现有同级敌人，保持在 ±20% 范围内
3. 强化效果避免无限叠加无上限，必须有 cap 或递减
4. 修改速度相关数值时注意对行动频率的指数级影响
5. 不要添加关卡间回血机制（修整节点已覆盖此需求）
