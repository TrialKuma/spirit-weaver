# 技能特效需求文档

**版本**：v1.0  
**范围**：玩家四职业主动技能 + 被动 + 追加技能  
**后续**：魂灵、敌人、UI、场景（第二阶段另开文档）

---

## 总体视觉方向

- **主风格**：灵魂仪式 + 霓虹秘术 + 暗色科幻面板
- **画面策略**：大面积深色背景，小面积高亮能量，重点元素用轮廓光和符文纹路建立层级
- **图形语言**：少用具象写实，多用剪影、徽章、几何法阵、方向性刀光、能量碎片
- **透明底**：所有特效资源均为透明背景 PNG

### 职业色板

| 职业 | 主色 | 辅色 | 高亮色 | 视觉关键词 |
|------|------|------|--------|-----------|
| 气宗 | #ff4d4d 赤红 | #ff8a65 橙焰 | #ffffff 白芯 | 灼赤魂火、内劲涡流、燃脉裂纹 |
| 剑圣 | #ffd700 金 | #90caf9 冰蓝 | #ffffff 白锋 | 金白锋芒、交叉刃痕、疾风切线 |
| 魔导 | #64b5f6 蓝 | #3399ff 深蓝 | #ffd54f 金弹 | 蓝青结晶、弹舱符印、脉冲炮口 |
| 判官 | #b388ff 紫 | #ffe0b2 暖白 | #e040fb 品红 | 黑白紫金、阴阳法盘、审判印轮 |

## 资源规格

- **静态图素**：128x128 或 256x256，透明底 PNG
- **战斗特效**：横向拼接 spritesheet PNG
  - 轻型：256x256/帧，4帧
  - 重型：384x384/帧，6帧
  - 终结型：384-512/帧，6-8帧

## 表现分层

| 层级 | 内容 | 实现方式 |
|------|------|----------|
| 起手动作 | 前冲/后座 | CSS 动画（保留） |
| Cut-in 喊话 | 技能名横幅 | 程序 + 职业色（保留） |
| 起手全局 | 震屏/闪屏 | 程序（保留） |
| **核心特效** | **斩击/弹体/法阵** | **spritesheet** |
| **命中爆点** | **冲击粒子/爆散** | **spritesheet** |
| 伤害数字 | 跳字 | 程序（保留） |
| 受击反馈 | 闪红/击退/血条抖动 | CSS（保留） |
| 状态提示 | BUFF/Debuff | 程序（保留） |

加粗项为资源化目标，其余保留程序实现。

## 一、气宗 (QiMaster)

### 1.1 轻击 light_strike

- **定位**：零消耗起手，平稳输出
- **标签**：Light, Melee
- **核心特效**：轻型直线斩击，短弧、薄刃、快消散
- **命中爆点**：轻型粒子爆散，少量火星
- **配色**：刀光白芯，边缘微带赤红
- **镜头**：无震屏，无闪屏
- **伤害数字**：小号，标准淡出
- **资源**：复用 `slash_light.png` + `impact_light.png`
- **复用**：全局模板，四职业轻击共用，程序 tint 换色

### 1.2 迅击 rapid_strike

- **定位**：中耗特技，内伤联动返气
- **标签**：Special, Melee
- **核心特效**：橙焰弧斩，比轻斩更宽更亮，带微弱焰尾拖痕
- **命中爆点**：标准粒子爆散，橙红配色
- **配色**：主体 #ff8a65 橙焰，白芯，边缘赤红
- **镜头**：无震屏，无闪屏
- **伤害数字**：标准大小
- **附加**：命中内伤目标返气时弹出 +2气 提示（程序）
- **资源**：**职业专属** `slash_special_qi.png`（6帧，256x256/帧）
- **复用**：气宗专属

### 1.3 崩山 devastate

- **定位**：高耗终极爆发，触发被动挂内伤
- **标签**：Heavy, Ultimate, Melee
- **核心特效**：大型赤红重斩，厚刃弧光，强烈发光，纵向撕裂感
- **命中爆点**：重型爆散，大量赤红火星 + 地面裂纹碎片
- **配色**：主体 #ff4d4d 赤红，白芯极亮，边缘暗红
- **镜头**：起手震屏（强度8，300ms）+ 闪屏（红0.25，80ms）
- **伤害数字**：大号，加长停留
- **附加**：命中后敌方出现内伤 debuff 图标（程序）
- **资源**：**职业专属** `slash_heavy_qi.png`（6帧，384x384/帧）+ `impact_heavy.png`
- **复用**：气宗专属斩击；命中爆点可换色共用

### 1.4 被动：内伤打击

- **触发**：消耗>=6气的技能命中后
- **视觉**：敌方框体短暂闪现裂纹/内出血标记
- **资源**：程序实现（红色脉冲 + BUFF 图标）

### 1.5 追加技能

- **追击**：复用 `slash_light.png` 金黄 tint
- **劲气**：复用 `slash_light.png` + 弹出推迟提示文字

## 二、剑圣 (ComboMaster)

### 2.1 疾风 quick_strike

- **定位**：2段速攻起手，叠疾风层
- **标签**：Light, MultiHit, Melee
- **核心特效**：首段标准轻斩；第2段小型快速斩痕
- **命中爆点**：每段轻型爆散，少量金色碎片
- **配色**：金白双色，#ffd700 金边 + 白芯
- **镜头**：无震屏，无闪屏
- **伤害数字**：小号，多段快速弹出
- **资源**：首段复用 `slash_light.png`（金色 tint）；后续段 `slash_multihit.png`
- **复用**：全局模板

### 2.2 连斩 combo_strike

- **定位**：概率追加连段（1-5 hit），攒连击资源
- **标签**：Light, MultiHit, Melee
- **核心特效**：首段轻斩；后续每段快速小斩痕，角度交替
- **命中爆点**：每段轻型爆散
- **配色**：与疾风相同，金白双色
- **镜头**：无震屏。5连斩时可加轻微震屏（程序判断）
- **伤害数字**：小号，密集连续弹出
- **附加**：每 hit 弹出 +1连击 跳字（程序）
- **资源**：首段 `slash_light.png`；后续段 `slash_multihit.png`
- **复用**：全局模板

### 2.3 终结技 finisher

- **定位**：消耗连击兑现伤害，单段重击终结
- **标签**：Heavy, Ultimate, Finisher, Melee
- **核心特效**：大型金色纵斩，锋芒感强，交叉刃痕残影，尾端碎散
- **命中爆点**：重型爆散，金色碎片 + 白光环扩散
- **配色**：主体 #ffd700 纯金，白芯极亮，边缘 #ff8f00 暗金
- **镜头**：起手震屏（强度8，300ms）+ 闪屏
- **伤害数字**：大号，加长停留
- **资源**：**职业专属** `slash_heavy_combo.png`（6帧，384x384/帧）+ `impact_ultimate.png`
- **复用**：剑圣专属斩击；`impact_ultimate.png` 全局共用

### 2.4 被动：疾风奥义

- **触发**：每次攻击叠1层疾风（满10层 +20%攻击）
- **视觉**：速度拖影/残影提示（低优先级）
- **资源**：程序实现（BUFF 图标 + 满层金色呼吸光 CSS）

### 2.5 追加技能

- **回旋斩**：复用 `slash_light.png` 金黄 tint
- **破绽**：复用 `slash_light.png` + 程序 debuff 图标

## 三、魔导 (ManaMaster)

### 3.1 射击 shoot（普通态）

- **定位**：无弹药时基础远程攻击
- **标签**：Ranged
- **核心特效**：小型蓝色能量弹，中等拖尾，线性飞行
- **命中爆点**：蓝色能量爆散，轻型
- **配色**：弹体 #64b5f6 蓝，白芯；拖尾蓝色渐淡
- **镜头**：无震屏。轻微后座（CSS 已有）
- **伤害数字**：标准大小
- **资源**：`projectile_normal.png`（4帧，128x128/帧）+ `impact_ranged.png`
- **复用**：魔导普通态通用

### 3.2 强化射击 shoot（有弹药态）

- **定位**：消耗1弹药，高倍率单发
- **标签**：Ranged, Ultimate
- **核心特效**：大型金色能量弹，强发光，密集拖尾
- **命中爆点**：金色能量爆散，中型
- **配色**：弹体 #ffd54f 金，白芯极亮；拖尾金色
- **镜头**：起手震屏（强度8，300ms）+ 闪屏。后座已有
- **伤害数字**：大号
- **资源**：`projectile_enhanced.png`（4帧，192x192/帧）+ `impact_ranged.png`（金色 tint）
- **复用**：魔导强化态通用

### 3.3 过载爆射 burst（普通态）

- **定位**：无弹药时重型远程，自带减速 debuff
- **标签**：Ranged, Heavy
- **核心特效**：红色不稳定能量弹，大尺寸，边缘闪烁/不规则，表达过载
- **命中爆点**：红色重型爆散
- **配色**：弹体 #ff4444 红，边缘不稳定红白闪烁
- **镜头**：无震屏（非 Ultimate）
- **伤害数字**：标准偏大
- **附加**：命中后玩家弹出减速提示 + 蓝色粒子（程序已有）
- **资源**：`projectile_overload.png`（6帧，192x192/帧）+ `impact_heavy.png`（红色 tint）
- **复用**：魔导过载专属

### 3.4 强化爆射 burst（有弹药态）

- **定位**：消耗3弹药，极高倍率
- **标签**：Ranged, Heavy, Ultimate
- **核心特效**：超大金色能量弹，最强发光，最密拖尾
- **命中爆点**：金色重型爆散，最大范围
- **配色**：#ffd54f 金主体，白芯极亮，边缘 #ff8f00 暗金
- **镜头**：起手震屏 + 闪屏
- **伤害数字**：大号，加长停留
- **资源**：复用 `projectile_enhanced.png` 程序放大 + `impact_ultimate.png`
- **复用**：复用强化弹体 + 程序增强

### 3.5 装填 reload

- **定位**：功能技，消耗魔力获弹药，无伤害
- **标签**：Reload, Ultimate
- **核心特效**：玩家周围能量汇聚/充填光环
- **命中爆点**：无
- **配色**：#64b5f6 蓝汇聚 -> #ffd54f 金闪
- **镜头**：起手震屏（Ultimate 触发），可考虑降低强度
- **伤害数字**：无。弹出 +3弹药 提示（程序）
- **资源**：`reload_charge.png`（4帧，256x256/帧）— 优先级低，可后做
- **复用**：魔导专属

### 3.6 被动：魔力充盈

- **触发**：消耗魔力叠盈能；攻击消耗1层回3魔力
- **视觉**：盈能叠层时微弱电弧闪烁；消耗时蓝色回流
- **资源**：程序实现（BUFF 图标 + 回魔跳字）

### 3.7 追加技能

- **余波**：复用 `projectile_enhanced.png` 金黄 tint
- **过载充能**：复用 `reload_charge.png` + 程序提示

## 四、判官 (BalanceMaster)

### 4.1 阳击 yang_strike

- **定位**：阳面推进，平衡 +3，行动值推进 15%
- **标签**：Yang, Melee
- **核心特效**：暖白光斩，从上至下直弧，纯净明亮，微弱上升光粒
- **命中爆点**：白色光粒爆散，少量金色碎片
- **配色**：主体 #ffe0b2 暖白，白芯，边缘 #ffcc80 暖金
- **镜头**：无震屏，无闪屏
- **伤害数字**：标准大小
- **附加**：行动值推进提示（程序）
- **资源**：**职业专属** `slash_yang.png`（6帧，256x256/帧）
- **复用**：判官阳系专属

### 4.2 烈阳击（阳极增强态）

- **定位**：阳极状态阳击增强，250%倍率，无视防御
- **标签**：同 Yang, Melee（极值增强版）
- **核心特效**：加强版阳斩，更宽更亮，强烈光芒扩散 + 金白能量裂痕
- **命中爆点**：白金色重型爆散
- **配色**：主体 #ffffff 纯白，边缘 #ffd700 金，极强发光
- **镜头**：命中时轻震屏（程序按伤害比例）
- **伤害数字**：大号
- **资源**：复用 `slash_yang.png` 程序放大 + 提亮。后期可升级独占
- **复用**：先复用 + 程序增强

### 4.3 阴击 yin_strike

- **定位**：阴面推进，平衡 -3，施加侵蚀 DOT
- **标签**：Yin, Melee
- **核心特效**：紫色暗弧斩，从下至上逆弧，侵蚀粒子拖尾，暗色雾气
- **命中爆点**：紫色暗粒子爆散，侵蚀扩散感
- **配色**：主体 #b388ff 紫，暗芯，边缘 #7c4dff 深紫
- **镜头**：无震屏，无闪屏
- **伤害数字**：标准大小
- **附加**：命中后敌方出现侵蚀 debuff 图标（程序）
- **资源**：**职业专属** `slash_yin.png`（6帧，256x256/帧）
- **复用**：判官阴系专属

### 4.4 深渊击（阴极增强态）

- **定位**：阴极状态阴击增强，180%倍率，深渊侵蚀 40% 2回合
- **标签**：同 Yin, Melee（极值增强版）
- **核心特效**：加强版阴斩，更宽更深，暗蚀扩散波纹 + 地面暗裂纹
- **命中爆点**：深紫黑色重型爆散，侵蚀雾气残留
- **配色**：主体 #7c4dff 深紫，#311b92 暗紫芯
- **镜头**：命中时轻震屏（程序按伤害比例）
- **伤害数字**：大号
- **附加**：敌方出现深渊侵蚀 debuff 图标
- **资源**：复用 `slash_yin.png` 程序加深 + 加大。后期可升级独占
- **复用**：先复用 + 程序增强

### 4.5 宣判 verdict

- **定位**：仅极值可用，平衡归零，落差爆发
- **标签**：Judgment, Ultimate, Melee
- **核心特效**：双刀交叉 X 斩，一道暖白 + 一道暗紫，交叉旋转撕裂，阴阳对冲
- **命中爆点**：阴阳双色爆散，白 + 紫交错碎片
- **配色**：阳道 #ffe0b2 暖白，阴道 #b388ff 紫，交叉点 #e040fb 品红闪爆
- **镜头**：起手震屏（强度8，300ms）+ 闪屏
- **伤害数字**：大号，加长停留
- **附加**：阴极宣判命中后敌方出现审判 debuff 图标
- **资源**：**职业专属** `slash_judgment.png`（8帧，384x384/帧）+ `impact_ultimate.png`
- **复用**：判官独占，不可复用

### 4.6 被动：阴阳之道

- **触发**：阳面增益 / 阴面减益 / 极值激活 / 反差加成
- **视觉**：极值激活时全屏短暂闪光（阳极白金 / 阴极深紫）
- **资源**：程序实现（BUFF 图标 + 极值闪屏 CSS）

### 4.7 追加技能

判官不设追加技能。
