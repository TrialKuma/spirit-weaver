# BD感重设计方案

**状态**：讨论中，未开始实现
**最后更新**：2026.3.22

---

## 核心问题

现有强化系统只有"数值变大"，没有build差异。每个职业4个专属强化都是线性数值叠加，所有路线终点都是"打更多伤害"，只是到达方式不同。

**根本原因**：每个职业只有"攻击打伤害"一种赢法。需要每个职业有"签名机制"作为第二赢法。

---

## 核心设计决策

### 1. 哈迪斯锤子模式

**不加新技能，通过强化改变现有技能行为。** 每个职业3个技能，每个技能有2-3种"锤子"变体方向。

规则：
- 同一个技能只能吃一个锤子，互斥
- 每个职业约6-9个锤子可选
- 每局碰到1-2次锤子机会，每次3选1
- 锤子单独给（不跟普通强化混），有仪式感
- 获取时机：待定（精英战后 / 特殊节点 / 铁匠铺）

分工：
- **锤子** = 定方向，改变技能形态，决定"这局怎么打"
- **普通强化** = 补数值，围绕锤子定的方向叠加成
- 锤子让普通强化有了方向感——这就是BD感的来源

### 2. 每个职业需要"签名机制"

不只是资源系统不同，输出方式/赢法也要不同：

| 职业 | 直接伤害（已有） | 签名机制（需要改/加） |
|------|----------------|---------------------|
| 气宗 | 崩山爆发 | 内伤叠层（从易伤改为可叠层可引爆的独立系统） |
| 剑圣 | 终结技爆发 | 疾风Style化（加衰减，变成需要维护的动态系统） |
| 魔导 | 强化射击 | 盈能主动化（从被动buff变成可花费的主动资源） |
| 判官 | 宣判爆发 | 待定（底子最好，阴阳系统天然有分支） |

### 3. 流程拉长到3幕

当前13节点太短，build来不及成型。

目标结构：
- 幕1：(3普通+1精英+1休息) + 幕Boss = ~6节点
- 幕2：(3普通+1精英+1休息) + 幕Boss = ~6节点
- 幕3：(3普通+1精英+1休息) + 最终Boss = ~6节点
- 总计约21节点

一局大概能拿：15个左右普通强化 + 2-3个锤子。这个量够build成型。

敌人填充优先级低——先用现有敌人撑起3幕（数值缩放），后面再补变体和新敌人。

## 气宗详细设计

### 基础改动

- **内伤从"易伤buff"改为可叠层系统**：每层不造成伤害，叠满后可被引爆造成高伤害
- **削弱崩山基础值**：从8气500%改为约6气350%，让基础状态是个中庸角色，靠BD强化分化

### 气宗锤子方向

**轻击（A技能，基础0气100%）**
- 「连环拳」：轻击变3段（每段40%），每段叠1层内伤 → 变成内伤流的叠层工具
- 「回气掌」：轻击命中后回复1气 → 变成气回转工具

**迅击（S技能，基础2气200%）**
- 「裂伤击」：迅击额外叠2层内伤，内伤目标额外+50%伤害 → 内伤流主力伤害技
- 「双迅」：迅击变为打2次（每次150%），消耗3气 → 纯直伤路线
- 「迅影」：迅击消耗改为0气，倍率降到100%，推进自身行动条20% → 高频轻量出手

**崩山（U技能，改后6气350%）**
- 「内伤引爆」：崩山引爆目标所有内伤层，每层+60%基础攻击力额外伤害 → 内伤流收割器
- 「蓄力崩山」：崩山可消耗所有气（≥6），每多1气+50%倍率，满气=850% → 一刀流
- 「碎山连」：崩山消耗降到4气，倍率降到200%，可连续使用 → 频繁小崩山回转流

### 气宗build路线

- **内伤流** = 连环拳 + 裂伤击 + 内伤引爆。叠层→引爆→再叠→再引爆
- **一刀流** = 回气掌 + 双迅 + 蓄力崩山。攒满气一刀850%
- **回转流** = 回气掌 + 迅影 + 碎山连。0气迅击+回气轻击+4气小崩山，快速循环

### 气宗待定问题

- 内伤层数上限？（5层？10层？影响引爆伤害上限和叠层节奏）
- 内伤引爆是崩山专属还是可以有其他引爆方式？
- 内伤引爆后层数清零还是消耗一部分？
- 普通强化怎么围绕锤子设计？

## 剑圣方向（待展开）

核心改动：**疾风加衰减**，从"叠了不掉"变成需要维护的Style系统。
- 不攻击就掉层
- 被打掉层
- 不同层数段有不同增益
- build围绕"怎么对待疾风层数"展开

可能的路线：
- 疾风维护流 = 保持高层数持续高增益
- 终结流 = 攒连击一刀终结
- 疾风燃烧流？ = 消耗所有疾风层释放大招（待讨论）

---

## 魔导方向（待展开）

核心方向：**哈迪斯模式，不加新技能，锤子改变现有3个技能的行为。**

"射击"的锤子可能方向：散射、蓄力、连射
"爆射"的锤子可能方向：待定
"装填"的锤子可能方向：超载装填、快速装填等

盈能从被动buff变主动可花费资源（满溢引爆等）

不做弹种系统（太复杂）。
