# 单局流程与强化系统

**基于**：设计文档 v3.0

---

## 整体结构

**节奏模型**：(2+1+1) × 3 + 1

```
阶段1: 普通战 → 普通战 → 精英战 → 修整
阶段2: 普通战 → 普通战 → 精英战 → 修整
阶段3: 普通战 → 普通战 → 精英战 → 修整
最终:  BOSS战
```

**节点总计**：6场普通战 + 3场精英战 + 3次修整 + 1场BOSS = 13个节点

### 完整流程

```
选择职业 → 魂灵祭坛（选魂灵）→ 阶段1 → 阶段2 → 阶段3 → 最终BOSS → 通关/失败
```

---

## 战斗节点

**普通战**：
- 从普通敌人池中随机匹配
- 胜利后获得 1 金币
- 战斗结束后获得一次强化（三选一，标准稀有度池）

**精英战**（每阶段第3场）：
- 从精英敌人池中匹配，拥有独特机制
- 胜利后获得 3 金币
- 战斗结束后获得一次强化（四选一，**高稀有度池**）

**最终BOSS**：
- 固定BOSS，拥有多阶段机制
- 胜利后获得 5 金币
- 通关即胜利

## 强化系统

### 强化节奏

- **普通战后**：三选一，标准稀有度池
- **精英/Boss后**：四选一，高稀有度池
- **可跳过**：玩家可以选择不拿强化

### 稀有度池

| 池类型 | R | SR | SSR |
|--------|---|-----|-----|
| 标准池（普通战后） | 70% | 25% | 5% |
| 高级池（精英/Boss后） | 40% | 40% | 20% |

### 升级规则

- 强化可重复出现并升级（仅数值类，`stackable: true`）
- 机制类强化仅出现一次
- 升级显示：`Lv.1 → Lv.2`
- 升级成长：`value = baseValue × (1 + 0.2 × (level-1))`
