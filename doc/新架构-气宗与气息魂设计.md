# 新架构设计：气宗 + 气息魂 + 连击魂

**状态**：设计定稿，准备实装
**日期**：2026.4.14（最后更新）

---

## 设计架构说明

**核心变更**：角色是"底盘"（技能+资源系统），魂灵提供被动和build方向。角色被动能力移至魂灵系统。

- 角色的锤子：改变技能形态（替换/新增追加）
- 魂灵的锤子：改变被动效果和魂灵攻击
- AV系数：**不做**全局技能AV系数。时间管理通过具体技能的推条效果实现（震破、飞驰步等）
- 速度比：维持 **3:1**（玩家300, 敌人~100）

**资源**：气息（0-10），初始满值，每回合回复2

| 类型 | 名称 | 消耗 | 倍率 | 效果 |

## 气息魂（被动型）

不走CTB条。所有效果自动触发。

### 被动——【内伤】

**触发条件**：玩家**单次消耗≥6点资源**时触发（不是累计）

> 跨职业规则：判官按翻转时触发（阴阳翻转那一刻才算）。魔导只算魔力消耗，弹药消耗不计入。

**触发效果**：
1. 魂灵对目标进行一次攻击（**100%** 倍率，基于玩家ATK）
2. 对目标附加【内伤】

**【内伤】基础效果**：目标下次受到的伤害提升20%，持续1次受击。被任何伤害消耗（玩家或魂灵攻击均可）。

### 被动分支（魂灵锤子）

内爆与重伤互斥。回流可与任何路线搭配。

#### 内爆（与重伤互斥）

- 内伤改为按**目标行动回合数**计算持续时间（不再按受击次数消耗）
- 内伤可叠加至3层（每次单次≥6消耗触发时叠1层）
- 叠满3层时：清除所有内伤层 → 触发爆发伤害（**300%**）
- 易伤幅度不随层数叠加（始终20%）
- 后续强化方向：爆发伤害提升

#### 重伤（与内爆互斥）

- 内伤易伤幅度提升至**60%**
- 后续强化方向：附加额外debuff（被暴击率↑、DEF↓、内伤持续次数↑等）

#### 回流（可与内爆或重伤搭配）

- **触发条件**：内伤被消耗时触发（无论被玩家攻击消耗还是被魂灵攻击消耗）
- **效果**：玩家获得+1 ATK，永久叠加，暂不设cap
- 后续强化方向：ATK提升幅度增加、附带速度提升、附带资源回复等

### 攻击技能

#### 旋风斩

**触发条件**：玩家**累计**消耗12点资源后触发（触发后重置计数器）

> 注意：旋风斩用累计计数器，与被动的"单次≥6"是两套独立的判定。

**效果**：对所有敌人造成3次伤害（每段 **100%** 倍率）

| 强化 | 效果 |
|------|------|
| 强化1 | 攻击段数+1（变4段） |
| 强化2 | 攻击伤害增加 |
| 强化3 | 触发所需累计量 -33%（12→8） |

#### 续剑

**触发条件**：玩家释放**特技类**技能时触发（迅击/铁山靠/闪击/迅连击）

**效果**：对目标造成伤害（**80%** 倍率）

| 强化 | 效果 |
|------|------|
| 强化1 | 倍率提升至 **120%** |
| 强化2 | 释放时为玩家回复1点施法资源 |
| 强化3 | 连续释放第3次时，转变为续剑·终（**160%** 倍率） |

> 续剑连续计数：连续释放特技时+1，释放其他类型技能时归零。

### 触发优先级（同一次行动内）

当玩家释放一个技能后，按以下顺序处理：

```
1. 玩家技能伤害结算
   └→ 如果目标有【内伤】→ 伤害吃易伤加成 → 消耗内伤
      └→ 如果有【回流】→ 内伤被消耗 → 玩家+1 ATK
2. 角色追加技能判定（连拳、震破、回击等）
3. 续剑判定（如果释放的是特技类）
4. 旋风斩累计计数器更新（+本次消耗量）
   └→ 如果累计≥12 → 触发旋风斩 → 重置计数器
5. 被动判定（本次消耗是否≥6）
   └→ 如果是 → 魂灵攻击（100%）→ 附加【内伤】
```

**关键顺序说明**：
- 步骤1先消耗已有内伤 → 步骤5才挂新内伤。同一次行动内不会出现"自己挂的内伤自己吃"
- 续剑（步骤3）在被动（步骤5）之前触发，所以续剑不会吃到本次被动挂的内伤
- 旋风斩（步骤4）在被动（步骤5）之前触发，所以旋风斩也不会吃到本次被动挂的内伤
- 回流在步骤1触发（内伤被玩家消耗时），不依赖魂灵攻击命中

## 连击魂（手操型）

走CTB行动条。魂灵行动到达时，玩家选择释放斩风或飞驰步。

**魂灵基础速度**：150

### 被动——【疾风】

**触发**：玩家每次行动后+1层【疾风】（包括追加技能触发时也+1层）

**效果**：每层+10速度（加给玩家），上限10层

**疾风消耗规则**：疾风只通过魂灵主动技能消耗（斩风-2层、飞驰步-5层），不会自然衰减。

### 被动分支（魂灵锤子）

| 分支 | 效果 | 后续强化方向 |
|------|------|-------------|
| **乘风** | 魂灵也受疾风加速，每层+5速度 | 速度值提升 |
| **裂刃** | 每3层疾风使玩家每段伤害额外附加 **60%** 倍率（独立扣防，不算命中） | 倍率提升 |
| **狂风** | 每累计获得5层疾风时，魂灵自动追加一次全体 **150%** 伤害 | 所需层数↓、倍率↑ |

### 攻击技能

> 魂灵走条到了行动时，玩家选择释放一个技能。也可以选择不释放（保留疾风）。

#### 主动：斩风

**消耗**：2层疾风

**效果**：对目标造成2段伤害（每段 **90%**，总180%）

| 强化 | 效果 |
|------|------|
| 强化1 | 造成3段伤害（每段90%，总270%） |
| 强化2 | 攻击后返还1层疾风 |
| 强化3 | 攻击后魂灵行动值前进30% |

#### 主动：飞驰步

**消耗**：5层疾风

**效果**：对目标造成1次伤害（**100%**），使玩家行动提前100%（立即再行动一次）

| 强化 | 效果 |
|------|------|
| 强化1 | 玩家下次攻击命中目标时额外获得3层疾风 |
| 强化2 | 额外使玩家攻击力提升20%（持续1次行动） |
| 强化3 | 玩家下次攻击命中目标时，使魂灵行动前进100% |

#### 追加：交错之斩

**触发条件**：魂灵和玩家交错攻击累计3次后可触发

> 交错计数规则：严格轮流。玩家攻击后如果上次是魂灵攻击→+1，反之亦然。连续同一方攻击不计数。即：玩家→魂灵→玩家 = 3次，魂灵→玩家→魂灵 = 3次。

**效果**：魂灵和玩家各自同时对目标追加1次攻击（各 **150%**，总300%）

| 强化 | 效果 |
|------|------|
| 强化1 | 两次攻击完成后，目标陷入【易伤】（受伤提升20%，持续1回合） |
| 强化2 | 技能伤害提升 |
| 强化3 | 释放后获得3层疾风 |

### 触发优先级

```
魂灵行动到达时：
1. 玩家选择：斩风 / 飞驰步 / 不行动
2. 魂灵技能伤害结算
3. 如果是飞驰步 → 玩家行动提前100%
4. 交错计数更新
   └→ 如果交错≥3 → 交错之斩触发 → 重置计数

玩家行动时：
1. 玩家技能伤害结算
2. +1层疾风
3. 角色追加技能判定（连拳等）
   └→ 每个追加触发也+1层疾风
4. 交错计数更新
```

## 两个魂灵对比

| | 气息魂 | 连击魂 |
|---|---|---|
| **类型** | 被动型（不走条，自动响应） | 手操型（走条，玩家选技能） |
| **核心机制** | 【内伤】debuff目标 | 【疾风】buff自身速度 |
| **鼓励节奏** | 攒资源→大额消耗(≥6)→触发 | 高频出手→叠疾风→选时机花 |
| **气宗配合** | 攒气→崩山爆发 | 频繁轻拳/迅击+追加蹭疾风 |
| **操作复杂度** | 低（自动） | 高（双线+疾风管理） |
| **build分化** | 内爆(爆发)/重伤(即时)/回流(长线) | 乘风(速度)/裂刃(附加伤害)/狂风(自动AOE) |
