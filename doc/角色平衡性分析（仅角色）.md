# Spirit Weaver 角色平衡性分析（仅角色部分）

**分析依据**：设计文档 v2.0 中的角色设计部分  
**分析范围**：仅分析4个角色，不包含魂灵系统

---

## 一、角色基础数据汇总

### 1.1 基础属性（所有角色相同）

| 属性 | 数值 |
|------|------|
| HP | 100 |
| ATK | 10 |
| DEF | 5 |
| SPD | 300 |

### 1.2 技能伤害倍率汇总

**气宗**：
- 轻击：**80%**
- 重击：**200%**
- 崩山：**400%**
- 内伤（被动）：**50%**（耗气>6的技能触发）

**剑圣**：
- 疾风：**60% × 2 = 120%**（2次打击）
- 连斩：**20% × 5 = 100%**（5次打击）
- 终结技：**100% + 20% × 连击数**（1次打击，10连击=300%）

**魔导**：
- 无弹药状态：
  - 射击：**80%**
  - 连射：**60% × 2 = 120%**
- 有弹药状态：
  - 强化射击：**150%**
  - 强化连射：**120% × 3 = 360%**（3次打击）

**判官**：
- 阳击/阴击：**100%**
- 翻转：**200% + 50% × |平衡值|**（5平衡值=450%）

## 二、资源系统分析

### 2.1 气宗（Qi Master）

**资源**：
- 名称：气 (Qi)
- 最大值：10
- 初始值：**10**（开局满资源）
- 颜色：红色

**资源回转**：
- 每回合开始：**+2气**
- 技能消耗：0/4/8气

**资源循环**：
- **轻击循环**：每回合80%伤害，无消耗
- **重击循环**：4回合一次重击（200%），平均50%/回合
- **崩山循环**：8回合一次崩山（450%含内伤），平均56.25%/回合

**被动技能**：
- 内伤：耗气>6的技能触发，造成50%攻击力伤害
- 只有崩山（8气）能触发，重击（4气）不能触发

---

### 2.2 剑圣（Combo Master）

**资源**：
- 名称：连击 (Combo)
- 最大值：10
- 初始值：**0**（开局无资源）
- 颜色：黄色

**资源回转**：
- 每行动：**+1连击**
- 技能消耗：0/0/全部连击

**资源循环**：
- **疾风循环**：每回合120%伤害（2hit），无消耗，+1连击
- **连斩循环**：每回合100%伤害（5hit），无消耗，+1连击
- **终结技循环**：10次行动后使用终结技（300%），平均30%/回合

**被动技能**：
- 速度叠加：每次攻击+1层速度，最高10层
- 每层速度+5，最高叠加到150速度
- 多hit技能不叠加多层（只+1层）

**速度叠加效果**：
- 初始速度：300
- 10层后速度：300 + 50 = 350（提升16.7%）
- AV变化：33.3 tick → 28.6 tick（行动频率提升16.7%）

---

### 2.3 魔导（Mana Master）

**资源**：
- 主资源：魔力 (Mana)，最大值10，初始值**10**
- 次资源：弹药 (Ammo)，最大值999，初始值**0**
- 颜色：蓝色

**资源回转**：
- 每行动：**+2魔力**（被动）
- 装填：**5魔力 → 5弹药**
- 消耗弹药：**额外回复2魔力**

**资源循环**：
- **无弹药循环**：每回合80%-120%伤害，+2魔力
- **装填循环**：
  1. 装填：消耗5魔力 → 获得5弹药
  2. 强化连射：消耗3弹药 → 360%伤害 + 回复6魔力
  3. 净收益：-5魔力 + 6魔力 = **+1魔力**（净赚）

**被动技能**：
- 魔力回复：每次行动回复2点魔力

---

### 2.4 判官（Balance Master）

**资源**：
- 名称：平衡 (Balance)
- 范围：**-5 到 +5**
- 初始值：**0**
- 颜色：紫色

**资源回转**：
- 每回合开始：**向0靠拢2点**（自动归中）
- 技能偏移：±2点

**资源循环**：
- **积攒阶段**：3回合达到±5平衡值
  - 阳击+2，阴击-2
  - 需要3次技能才能达到±5
- **翻转爆发**：450%伤害（200% + 50%×5）
- **平均伤害**：450% ÷ 3回合 = **150%/回合**

**被动技能**：
- 平衡之力：
  - 负值时：降低敌方攻击和防御
  - 正值时：提升自己全属性（**每点平衡值+10%属性**）
  - 5点平衡值 = +50%全属性

**归中机制**：
- 每回合归中2点
- 5点平衡值 → 3回合后归零
- 操作窗口期：**3回合**

## 三、伤害输出对比分析

### 3.1 单次技能最高伤害

| 角色 | 最高伤害技能 | 伤害倍率 | 资源消耗 | 资源效率 |
|------|------------|---------|---------|---------|
| **气宗** | 崩山+内伤 | **450%** | 8气 | 56.25%/气 |
| **剑圣** | 终结技（10连击） | **300%** | 10连击 | 30%/连击 |
| **魔导** | 强化连射 | **360%** | 3弹药 | 120%/弹药 |
| **判官** | 翻转（5平衡） | **450%** | 0资源 | ∞ |

**问题**：
- ⚠️ **剑圣终结技伤害明显偏低**（300% vs 其他450%/360%）
- ⚠️ **剑圣资源效率最低**（30%/连击 vs 其他56%/120%）

## 五、被动技能效果分析

### 5.1 气宗 - 内伤

**效果**：耗气>6的技能触发，造成50%攻击力伤害

**触发条件**：
- 只有崩山（8气）能触发
- 重击（4气）不能触发

**评价**：
- ⚠️ 触发条件较严格，只有能量技能触发
- 建议：降低阈值至4气，让重击也能触发

---

### 5.2 剑圣 - 速度叠加

**效果**：每次攻击+1层速度，最高10层，每层+5速度

**速度提升**：
- 初始：300
- 10层后：350（+16.7%）
- AV变化：33.3 → 28.6 tick

**评价**：
- ✅ 提升行动频率，符合"多动症"设计
- ⚠️ 但提升幅度较小（16.7%），可能需要更多层数或更高加成

---

### 5.3 魔导 - 魔力回复

**效果**：每次行动回复2点魔力

**评价**：
- ✅ 保证资源循环顺畅
- ✅ 配合装填循环，形成资源闭环

---

### 5.4 判官 - 平衡之力

**效果**：
- 负值：降低敌方攻击和防御
- 正值：提升自己全属性（每点+10%）

**属性提升**（5点平衡值）：
- 全属性+50%（HP、ATK、DEF、SPD）
- HP：100 → 150
- ATK：10 → 15
- DEF：5 → 7.5
- SPD：300 → 450

**评价**：
- ✅ 属性提升显著，但需要维持平衡值
- ⚠️ 归中速度过快（3回合归零），难以维持

## 六、核心平衡性问题

### 🔴 严重问题

1. **剑圣终结技伤害偏低**
   - **设计文档数值**：100% + 20%×连击数（10连击=300%）
   - **对比**：气宗450%，判官450%，魔导360%
   - **建议**：提升至 **150% + 25%×连击数**（10连击=400%）

2. **剑圣平均伤害输出最低**
   - **终结技循环**：平均30%/回合
   - **对比**：气宗56%，魔导72%，判官150%
   - **问题**：需要10次行动才能用终结技，但伤害却最低

3. **剑圣资源积攒最慢**
   - **需要10回合**才能满连击
   - **对比**：判官3回合，气宗开局满资源

### 🟡 中等问题

4. **判官平衡值归中过快**
   - **每回合归中2点**，5点平衡值3回合就归零
   - **操作窗口期短**：玩家需要在3回合内完成积攒和翻转
   - **建议**：归中速度改为 **1点/回合**

5. **判官平均伤害偏高**
   - **翻转循环**：平均150%/回合
   - **对比**：其他角色30%-72%
   - **但需要3回合准备**，实际可能合理

6. **魔导装填循环效率过高**
   - **5魔力→5弹药→360%伤害+6魔力**（净赚1魔力）
   - **可能形成无限循环**
   - **建议**：装填消耗提升至6魔力，或弹药生成降至4发

7. **气宗内伤触发条件过严**
   - **只有崩山（8气）能触发**，重击（4气）不能触发
   - **建议**：降低阈值至4气

### 🟢 轻微问题

8. **剑圣速度叠加提升幅度较小**
   - **10层后仅提升16.7%速度**
   - **建议**：提升每层加成至+10速度，或增加最大层数

## 七、平衡性调整建议

### 7.1 剑圣调整（优先级最高）

**终结技伤害**：
```
当前：100% + 20%×连击数（10连击=300%）
建议：150% + 25%×连击数（10连击=400%）
```

**或改为多hit机制**：
```
基础150%，每连击+1hit，每hit 50%伤害
10连击 = 11hit × 50% = 550%总伤害
```

**速度叠加**：
```
当前：每层+5速度，最高10层（+50速度）
建议：每层+10速度，最高10层（+100速度）
或：每层+5速度，最高20层（+100速度）
```

### 7.2 判官调整

**平衡归中速度**：
```
当前：每回合归中2点
建议：每回合归中1点
```

### 7.3 魔导调整

**装填成本**：
```
当前：5魔力→5弹药
建议：6魔力→5弹药
或：5魔力→4弹药
```

### 7.4 气宗调整

**内伤触发条件**：
```
当前：耗气>6
建议：耗气≥4
```

## 八、总结

### 8.1 角色强度排名（基于设计文档）

1. **判官**：平均伤害最高（150%/回合），但需要准备时间
2. **魔导**：平均伤害较高（72%/回合），循环顺畅
3. **气宗**：平均伤害中等（56%/回合），开局优势大
4. **剑圣**：平均伤害最低（30%/回合），需要调整

### 8.2 优先级建议

**第一优先级**（必须调整）：
1. 剑圣终结技伤害提升
2. 剑圣资源积攒速度优化

**第二优先级**（建议调整）：
3. 判官归中速度调整
4. 魔导装填成本调整
5. 气宗内伤触发条件调整
