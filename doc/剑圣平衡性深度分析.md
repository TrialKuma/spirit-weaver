# 剑圣平衡性深度分析（基于新机制）

**分析依据**：设计文档 v2.0（已更新：每次打击命中增加连击点）  
**重点**：考虑速度叠加带来的多动效果，分析长时间战斗的总输出

---

## 一、新机制理解

### 1.1 资源积攒机制（已更新）

**旧机制**：每行动+1连击点  
**新机制**：**每次打击命中+1连击点**

**技能打击数**：
- 疾风：2次打击 → **+2连击**
- 连斩：5次打击 → **+5连击**
- 终结技：1次打击 → 消耗所有连击

**积攒速度对比**：
- **旧机制**：需要10次行动才能满连击
- **新机制**：
  - 使用连斩：2次连斩 = 10次打击 = **10连击**（2回合满）
  - 使用疾风：5次疾风 = 10次打击 = **10连击**（5回合满）

---

## 二、速度叠加机制分析

### 2.1 速度叠加效果

**被动技能**：
- 每次攻击+1层速度（多hit技能不叠加多层）
- 每层速度+5
- 最高10层，最高叠加到150速度

**速度变化**：
- 初始速度：300 → AV = 33.3 tick
- 1层：305 → AV = 32.8 tick
- 5层：325 → AV = 30.8 tick
- 10层：350 → AV = 28.6 tick

**行动频率提升**：
- 初始：每33.3 tick行动一次
- 10层后：每28.6 tick行动一次
- **提升幅度**：16.7%（约1.17倍）

---

## 三、连击积攒速度分析（新机制）

### 3.1 使用连斩（最快积攒）

**连斩**：5次打击，每次+1连击 = +5连击

**积攒到10连击**：
- 第1回合：连斩 → +5连击（5/10）
- 第2回合：连斩 → +5连击（10/10，满）

**结论**：**2回合即可满连击**（vs 旧机制10回合）

---

### 3.2 使用疾风（较慢积攒）

**疾风**：2次打击，每次+1连击 = +2连击

**积攒到10连击**：
- 需要5次疾风 = 10次打击 = 10连击
- 需要5回合

**结论**：5回合满连击（vs 旧机制10回合）

## 四、长时间战斗总输出分析

### 4.1 战斗时间轴模拟（考虑速度叠加）

**假设**：30回合战斗（约1000 tick，考虑速度叠加后实际行动次数更多）

**速度叠加时间轴**：
- 回合1-2：速度300，AV=33.3，行动频率=1.0x
- 回合3-5：速度310-320，AV=32.3-31.3，行动频率≈1.03x
- 回合6-10：速度325-340，AV=30.8-29.4，行动频率≈1.08x

**3回合循环输出**：

- **总计**：500% / 3回合 = **166.7%/回合**

**考虑速度叠加**（10层后1.17倍频率）：
- 实际约 **195%/回合**

## 五、平衡性结论

### 5.1 新机制下的剑圣强度

**连击积攒速度**：
- ✅ **大幅提升**：从10回合降至2-5回合
- ✅ **连斩策略最优**：2回合即可满连击

**长时间总输出**：
- ✅ **连斩循环**：约195%/回合（考虑速度叠加）
- ✅ **疾风循环**：约175.5%/回合（考虑速度叠加）
- ✅ **明显高于其他角色**：
  - 气宗：56.25%/回合
  - 魔导：72%/回合
  - 判官：150%/回合

**结论**：
- ⚠️ **剑圣在长时间战斗中输出最高**（195%/回合）
- ⚠️ **可能过强**，需要调整

---

### 5.2 平衡性问题

**问题1：连击积攒过快**
- 连斩2回合即可满连击，节奏过快
- 建议：降低连击获取量，或提高连击上限

**问题2：终结技伤害可能偏低**
- 虽然总输出高，但单次终结技伤害（300%）仍低于其他角色（450%）
- 但考虑到积攒速度快，可能合理

**问题3：速度叠加提升幅度**
- 10层后仅提升16.7%，可能不够明显
- 但结合新机制，总输出已经很高

## 六、调整建议

### 6.1 连击积攒调整

**方案A：降低连击获取量**
```
每次打击命中+0.5连击（或每2次打击+1连击）
- 连斩：5hit → +2.5连击（或+2连击）
- 疾风：2hit → +1连击
- 需要4-5回合才能满连击
```

**方案B：提高连击上限**
```
连击上限：10 → 15或20
- 保持积攒速度，但需要更多连击才能用终结技
- 终结技伤害：100% + 20%×连击数（15连击=400%，20连击=500%）
```

**方案C：终结技消耗调整**
```
终结技消耗：全部连击 → 消耗10连击（固定）
- 允许保留部分连击，但终结技伤害固定
- 终结技伤害：提升至400%或450%
```

---

### 6.2 终结技伤害调整

**当前**：100% + 20%×连击数（10连击=300%）

**建议**：
- **方案A**：150% + 25%×连击数（10连击=400%）
- **方案B**：200% + 30%×连击数（10连击=500%）
- **方案C**：基础200%，每连击+1hit，每hit 50%伤害（10连击=11hit×50%=550%）

---

### 6.3 速度叠加调整（可选）

**当前**：每层+5速度，最高10层（+50速度）

**建议**：
- **方案A**：每层+10速度，最高10层（+100速度，提升33%）
- **方案B**：每层+5速度，最高20层（+100速度，提升33%）
- **方案C**：保持当前，但降低其他角色的输出

## 七、最终评价

### 7.1 新机制的优势

✅ **连击积攒速度快**：2-5回合即可满连击  
✅ **长时间输出高**：195%/回合（连斩循环）  
✅ **符合"多动症"设计**：速度叠加+多hit技能  
✅ **玩法更有趣**：需要选择积攒策略

### 7.2 需要调整的地方

⚠️ **连击积攒可能过快**：2回合满连击节奏太快  
⚠️ **长时间输出可能过高**：195%/回合 vs 其他角色56%-150%  
⚠️ **终结技单次伤害偏低**：300% vs 其他角色450%

### 7.3 建议优先级

**第一优先级**：
1. 测试新机制下的实际战斗体验
2. 根据测试结果调整连击积攒速度或终结技伤害

**第二优先级**：
3. 考虑提高连击上限或调整终结技消耗机制
4. 微调速度叠加效果
