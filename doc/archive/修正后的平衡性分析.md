## 一、机制理解修正

### 1.1 剑圣连击积攒机制

**正确理解**：
- **每次行动**：基础+1连击
- **疾风技能**：额外+1连击（所以疾风一次+2连击）
- **连斩技能**：3次打击，每次+1连击 = +3连击（加上行动基础+1 = 总共+4连击？）

**重新理解**（按用户说明）：
- 每次行动自然+1连击
- 疾风：2次打击，每次+1连击 = +2连击（加上行动基础+1 = 总共+3连击？）
- 或者：疾风额外+1，所以疾风一次 = 行动+1 + 疾风额外+1 = +2连击

**按用户说明**："疾风每次行动自然会+1，用疾风是额外+1，所以一直疾风的话5次行动就满"
- 疾风一次 = +2连击（行动+1 + 疾风额外+1）
- 5次疾风 = 5×2 = 10连击（满）

---

### 1.2 气宗【回转】机制

**正确理解**：
- 轻击：获得【回转】
- 持有【回转】时，下次攻击命中消耗【回转】回复2气
- **应该搭配迅击用**：轻击（100%）→ 迅击（200%，消耗【回转】回复2气）

**循环**：
- 轻击（100%，获得【回转】）→ 迅击（200%，消耗【回转】回复2气，消耗2气）
- 净消耗：0气
- 总伤害：300%

---

### 1.3 魔导装填循环

**资源流转**：
- 装填：6魔力 → 3发弹药
- 强化爆射：3弹药 → 伤害 + 回复6魔力
- 需要检查资源是否断档

---

### 1.4 判官机制

**平衡偏移**：+3/-3（不是+2/-2）  
**平衡加成**：每点+5%（不是+10%）  
**翻转CD**：2回合冷却

## 二、重新计算各职业输出

### 2.1 剑圣输出循环（修正）

**积攒阶段**（叠【疾风】）：
- 疾风×5：每次+2连击（行动+1 + 疾风额外+1）
- 5次疾风 = 10连击（满）+ 5层【疾风】

**满层后循环**（【疾风】10层，+20%攻击力，+100速度）：
- 连斩：3次打击，每次+1连击 = +3连击（加上行动+1 = +4连击？）
- 或者：连斩3次打击 = +3连击（行动基础+1已包含？）

**按文档**："每次打击命中都会增加连击点"
- 连斩：3次打击 = +3连击
- 但每次行动也会+1，所以连斩一次 = +4连击？

**简化理解**（按用户说明）：
- 疾风：行动+1 + 疾风额外+1 = +2连击
- 连斩：行动+1 + 3次打击 = +4连击

**循环**（积攒6连击用终结技）：
- 连斩×1.5：需要1.5次连斩 = 6连击
- 或者：连斩×2 = 8连击（超过6，浪费2）

**终结技**（消耗6连击）：
- 100% + 50% × 6 = **400%**

**满层【疾风】后**（+20%攻击力）：
- 连斩：30% × 1.2 = 36% × 3 = **108%**
- 终结技：400% × 1.2 = **480%**

**循环**（3回合，考虑速度叠加）：
- 连斩×2：108% × 2 = 216%（+8连击，消耗6用终结技）
- 终结技：480%
- **总计**：696% / 3回合 = **232%/回合**

**考虑速度叠加**（+100速度，行动频率提升33%）：
- 实际约 **309%/回合**

### 2.2 气宗输出循环（修正）

**轻击+迅击循环**：
- 轻击（100%，获得【回转】）→ 迅击（200%，消耗【回转】回复2气，消耗2气）
- 净消耗：0气
- 总伤害：300%
- **2回合循环**：**150%/回合**

**崩山+内伤循环**：
- 回合1：崩山（800% + 内伤标记）
- 回合2-3：轻击×2（100% + 100%内伤 = 200% × 2 = 400%）
- 回合4：迅击（200% + 100%内伤 = 300%）
- **4回合总输出**：800% + 400% + 300% = **1500%**
- **平均**：**375%/回合**

**但需要考虑资源**：
- 崩山消耗8气，每回合+2气
- 需要4回合才能回满8气
- 所以实际循环可能是：崩山 → 轻击×4（积攒气）→ 崩山

### 2.3 魔导装填循环（修正）

**装填循环**：
- 装填：6魔力 → 3发弹药
- 强化爆射：3弹药 → 伤害 + 回复6魔力

**资源检查**：
- 初始：10魔力
- 装填：-6魔力 = 4魔力，+3弹药
- 强化爆射：-3弹药 = 0弹药，+6魔力 = 10魔力
- **循环顺畅**，无断档

**强化爆射伤害**（按表格200%×3，按描述250%×3）：
- 按表格：200% × 3 = **600%**
- 第三次50%概率，期望：200% + 200% + 100% = **500%**

**【盈能】状态下**（攻击力+20%）：
- 强化爆射：500% × 1.2 = **600%**（期望）

**循环**（2回合）：
- 装填：6魔力 → 3弹药 + 3层【盈能】
- 强化爆射：600%（消耗3弹药，+6魔力）
- **2回合总输出**：600%
- **平均**：**300%/回合**

### 2.4 判官输出循环（修正）

**平衡偏移**：+3/-3  
**平衡加成**：每点+5%（5点 = +25%）  
**翻转CD**：2回合

**翻转循环**（5点平衡值）：
- 阳击×2：+6平衡（达到+5上限，浪费1）
- 或者：阳击×1 + 等待1回合（归中-2）→ 阳击×1 = +5平衡
- 翻转：200% + 50% × 5 = **450%**

**考虑属性加成**（+5平衡值，+25%全属性）：
- 翻转前：ATK × 1.25
- 翻转伤害：450% × 1.25 = **562.5%**

**循环**（考虑翻转CD 2回合）：
- 积攒阶段：2回合（阳击×2）
- 翻转：1回合（450% × 1.25 = 562.5%）
- 冷却：2回合（等待或使用其他技能）
- **5回合总输出**：200% + 562.5% = **762.5%**
- **平均**：**152.5%/回合**

**但翻转后平衡值变-5，需要重新积攒**
