# 最终平衡性计算与建议（修正版）

**分析依据**：设计文档 v2.0（最新版本）  
**关键理解**：
- **连击点**：只有每次打击命中才给连击点，行动本身不给
- **【疾风】**：每次行动（发动攻击）给【疾风】层数，不是连击点

---

## 一、机制正确理解

### 1.1 剑圣连击积攒机制（修正）

**正确理解**：
- **连击点**：每次打击命中+1连击
  - 疾风：2次打击 = +2连击
  - 连斩：3次打击 = +3连击
  - 终结技：1次打击 = +1连击
- **【疾风】层数**：每次行动（发动攻击）+1层【疾风】
  - 疾风：+1层【疾风】
  - 连斩：+1层【疾风】
  - 终结技：+1层【疾风】

**积攒速度**：
- 一直用疾风：每次+2连击，**5次疾风 = 10连击（满）**
- 一直用连斩：每次+3连击，**4次连斩 = 12连击（超过10，浪费2）**

## 二、各职业输出循环计算

### 2.1 剑圣输出循环（修正）

**积攒阶段**（叠【疾风】）：
- 疾风×5：每次+2连击 = 10连击（满）+ 5层【疾风】
- 伤害：80% × 5 = **400%**

**满层后循环**（【疾风】10层，+20%攻击力，+100速度）：
- 连斩：3次打击 = +3连击，伤害 30% × 3 = 90%
- 满层【疾风】加成：90% × 1.2 = **108%**
- 连斩×2：108% × 2 = 216%（+6连击）
- 终结技（消耗6连击）：100% + 50% × 6 = **400%**
- 满层【疾风】加成：400% × 1.2 = **480%**

**循环**（3回合）：
- 连斩×2：216%（+6连击）
- 终结技：480%
- **总计**：696% / 3回合 = **232%/回合**

**考虑速度叠加**（+100速度，从300→400，行动频率提升33%）：
- 实际约 **309%/回合**

---

### 2.2 气宗输出循环

**轻击+迅击循环**：
- 轻击（100%，获得【回转】）→ 迅击（200%，消耗【回转】回复2气，消耗2气）
- 净消耗：0气
- 总伤害：300%
- **2回合循环**：**150%/回合**

**崩山+内伤循环**：
- 回合1：崩山（800% + 内伤标记）
- 回合2-3：轻击×2（100% + 100%内伤 = 200% × 2 = 400%）
- 回合4：迅击（200% + 100%内伤 = 300%）
- **4回合总输出**：800% + 400% + 300% = **1500%**
- **平均**：**375%/回合**

**但需要考虑资源**：
- 崩山消耗8气，每回合+2气
- 需要4回合才能回满8气
- 实际循环：崩山 → 轻击×4（积攒气）→ 崩山
- **8回合循环**：800% + 400%（内伤加成） = **1200%**
- **平均**：**150%/回合**

**或者轻击+迅击循环**（无资源消耗）：
- **150%/回合**

---

### 2.3 魔导输出循环

**装填循环**：
- 装填：6魔力 → 3发弹药 + 3层【盈能】
- 强化爆射：3弹药 → 伤害（期望625%）+ 消耗弹药

**资源检查**：
- 初始：10魔力
- 装填：-6魔力 = 4魔力，+3弹药 + 3层【盈能】
- 【盈能】状态下：每次行动回复3点魔力
- 强化爆射：-3弹药 = 0弹药，伤害625%（期望）

**问题**：消耗弹药后，魔力如何回复？
- 文档说"消耗弹药时，额外回复2点魔力"，但这条在资源回转逻辑中被删除了
- 如果只有【盈能】状态回复3点魔力，那么：
  - 装填后：4魔力
  - 强化爆射：+3魔力（【盈能】状态）= 7魔力
  - 需要再等1回合才能装填（+3魔力 = 10魔力）

**循环**（3回合）：
- 装填：6魔力 → 3弹药 + 3层【盈能】
- 强化爆射：625%（期望）
- 等待1回合：+3魔力（【盈能】状态）
- **3回合总输出**：625%
- **平均**：**208.3%/回合**

**如果消耗弹药时额外回复2点魔力**：
- 装填：6魔力 → 3弹药 + 3层【盈能】
- 强化爆射：625% + 6魔力（【盈能】3点 + 消耗弹药2点）= 10魔力
- **2回合循环**：625%
- **平均**：**312.5%/回合**

---

### 2.4 判官输出循环

**平衡偏移**：+3/-3  
**平衡加成**：每点+5%（5点 = +25%）  
**翻转CD**：2回合

**翻转循环**（5点平衡值）：
- 阳击×2：+6平衡（达到+5上限，浪费1）
- 或者：阳击×1 + 等待1回合（归中-2）→ 阳击×1 = +5平衡
- 翻转：200% + 50% × 5 = **450%**

**考虑属性加成**（+5平衡值，+25%全属性）：
- 翻转前：ATK × 1.25
- 翻转伤害：450% × 1.25 = **562.5%**

**循环**（考虑翻转CD 2回合）：
- 积攒阶段：2回合（阳击×2，200%）
- 翻转：1回合（562.5%）
- 冷却：2回合（等待或使用其他技能，200%）
- **5回合总输出**：200% + 562.5% + 200% = **962.5%**
- **平均**：**192.5%/回合**

## 五、数值调整建议

### 5.1 剑圣调整（降低输出）

**方案A：降低终结技伤害**
```
当前：100% + 50% × 连击数（6连击 = 400%）
建议：100% + 30% × 连击数（6连击 = 280%）
```

**方案B：提高终结技消耗**
```
当前：消耗6连击
建议：消耗8连击或10连击
```

**方案C：降低【疾风】攻击力加成**
```
当前：满层+20%攻击力
建议：满层+10%攻击力
```

**推荐**：方案A + 方案C
- 终结技：100% + 30% × 连击数（6连击 = 280%）
- 【疾风】满层：+10%攻击力
- 调整后输出：约 **220%/回合**

---

### 5.2 魔导调整（降低输出）

**方案A：降低强化爆射伤害**
```
当前：250% + 250% + 250%（50%概率）= 625%（期望）
建议：200% + 200% + 200%（50%概率）= 500%（期望）
```

**方案B：提高装填消耗**
```
当前：6魔力 → 3弹药
建议：8魔力 → 3弹药
```

**方案C：降低【盈能】攻击力加成**
```
当前：+20%攻击力
建议：+10%攻击力
```

**推荐**：方案A + 方案C
- 强化爆射：200% + 200% + 200%（50%概率）= 500%（期望）
- 【盈能】：+10%攻击力
- 调整后输出：约 **250%/回合**

### 5.4 判官调整（提升输出）

**方案A：提升翻转基础伤害**
```
当前：200% + 50% × |平衡值|
建议：250% + 50% × |平衡值|
```

**方案B：提升平衡值加成**

**方案C：降低翻转CD**
