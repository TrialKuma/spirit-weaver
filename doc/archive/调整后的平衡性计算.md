## 一、调整1：魔导【盈能】基于层数

### 1.1 新机制

**【盈能】层数**：
- 每消耗2点魔力，获得1层【盈能】
- 每层【盈能】提升8%攻击力
- 装填消耗6魔力 = 3层【盈能】= +24%攻击力

### 1.2 输出计算（修正）

**装填循环**：
- 装填：6魔力 → 3弹药 + 3层【盈能】（+24%攻击力）
- 强化爆射：250% × 1.24 = 310% × 3 = 930%（期望，第三次50%概率）
- 实际期望：310% + 310% + 155% = **775%**

**最优循环**（爆射打空）：
- 装填：6魔力 → 3弹药 + 3层【盈能】（+24%攻击力）
- 强化爆射：775%
- 装填：6魔力 → 3弹药 + 3层【盈能】（+24%攻击力）
- 强化爆射：775%
- 射击：80% × 1.24 = 99.2%
- 等待1回合
- **5回合总输出**：775% + 775% + 99.2% = 1649.2%
- **平均**：**329.8%/回合**

**如果积攒更多【盈能】层数**：
- 装填×2：12魔力 → 6弹药 + 6层【盈能】（+48%攻击力）
- 强化爆射：250% × 1.48 = 370% × 3 = 1110%（期望）
- 实际期望：370% + 370% + 185% = **925%**
- 但需要更多回合积攒，可能不如直接循环

## 三、调整3：气宗内伤改成"此次伤害提升50%"

### 3.1 新机制

**内伤效果**：改成"此次伤害提升50%"
- 崩山附加内伤标记
- 下次攻击（包括崩山）伤害提升50%

### 3.2 新循环：崩山可以吃到内伤加成

**循环**：崩山（剩2）→ 轻击迅击（剩6）→ 轻击（8）→ 崩山（吃到内伤+50%）

**详细计算**：
- 回合1：崩山（800% + 内伤标记），消耗8气，剩余2气
- 回合2：轻击（100%），获得【回转】，剩余4气（+2自然回复）
- 回合3：迅击（200%），消耗【回转】回复2气，消耗2气，剩余4气（+2自然回复）
- 回合4：轻击（100%），获得【回转】，剩余6气（+2自然回复）
- 回合5：崩山（800% × 1.5 = 1200%，吃到内伤+50%），消耗8气，剩余0气（+2自然回复）
- 回合6：轻击（100%），获得【回转】，剩余2气（+2自然回复）
- 回合7：迅击（200%），消耗【回转】回复2气，消耗2气，剩余2气（+2自然回复）
- 回合8：轻击（100%），获得【回转】，剩余4气（+2自然回复）
- 回合9：崩山（800% × 1.5 = 1200%，吃到内伤+50%），消耗8气，剩余-2气（+2自然回复）= 0气

**5回合循环**（崩山×2）：
- 崩山1：800%
- 轻击+迅击：300%
- 轻击：100%
- 崩山2：1200%（吃到内伤）
- **总计**：2400% / 5回合 = **480%/回合**

**或者考虑更精确的循环**：
- 崩山（800% + 内伤）→ 轻击迅击（300%）→ 轻击（100%）→ 崩山（1200%）
- **4回合循环**：800% + 300% + 100% + 1200% = 2400%
- **平均**：**600%/回合**

**但需要考虑资源**：
- 崩山后剩余：2气
- 轻击迅击：消耗0气，剩余4气（+2自然回复）
- 轻击：消耗0气，剩余6气（+2自然回复）
- 崩山：消耗8气，剩余0气（+2自然回复）

**资源检查**：
- 回合1：崩山，剩余2气
- 回合2：轻击，剩余4气（+2）
- 回合3：迅击，剩余4气（消耗2，回复2，+2）
- 回合4：轻击，剩余6气（+2）
- 回合5：崩山，剩余0气（消耗8，+2）

**4回合循环可行**：
- **平均输出**：**600%/回合**

**如果50%太高，降低到30%**：
- 崩山2：800% × 1.3 = 1040%
- **4回合循环**：800% + 300% + 100% + 1040% = 2240%
- **平均**：**560%/回合**
