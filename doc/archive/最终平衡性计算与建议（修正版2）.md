## 一、气宗输出循环（修正）

### 1.1 轻击+迅击循环

**机制**：
- 轻击（100%，获得【回转】）→ 迅击（200%，消耗【回转】回复2气，消耗2气）
- 净消耗：0气
- 总伤害：300%（含内伤加成）

### 1.2 崩山循环（修正）

**循环**：崩山 → (轻击-迅击) × 2 → 崩山

**详细计算**：
- 回合1：崩山（800% + 内伤标记），消耗8气
- 回合2：轻击（100% + 100%内伤 = 200%），获得【回转】
- 回合3：迅击（200% + 100%内伤 = 300%），消耗【回转】回复2气，消耗2气
- 回合4：轻击（100% + 100%内伤 = 200%），获得【回转】
- 回合5：迅击（200% + 100%内伤 = 300%），消耗【回转】回复2气，消耗2气

**资源检查**：
- 崩山后剩余：2气
- 每回合自然回复：+2气
- 回合2：2+2=4气
- 回合3：4气（迅击消耗2气，回复2气 = 4气）
- 回合4：4+2=6气
- 回合5：6气（迅击消耗2气，回复2气 = 6气）
- 回合6：6+2=8气（可以崩山）

**5回合循环输出**：
- 崩山：800%
- 轻击+迅击×2：(200% + 300%) × 2 = 1000%
- **总计**：1800% / 5回合 = **360%/回合**

**或者6回合循环**（更准确）：
- 崩山：800%
- 轻击+迅击×2：1000%
- 等待1回合：轻击（200%）
- **总计**：2000% / 6回合 = **333.3%/回合**

## 二、魔导输出循环（修正）

### 2.1 资源流转逻辑

**第一次装填**：
- 初始：10魔力
- 装填：-6魔力 = 4魔力，+3弹药 + 3层【盈能】

**第一次强化爆射**：
- 消耗3弹药，伤害625%（期望）
- 【盈能】状态下回复3魔力：4+3=7魔力

**第二次装填**：
- 装填：-6魔力 = 1魔力，+3弹药 + 3层【盈能】

**选择A：打单发回魔力**
- 强化射击：180%伤害，消耗1弹药，剩余2弹药
- 【盈能】状态下回复3魔力：1+3=4魔力
- 再等1回合：+3魔力 = 7魔力
- 再等1回合：+3魔力 = 10魔力（可以装填）

**选择B：再次爆射打空**
- 强化爆射：625%伤害，消耗3弹药，剩余0弹药
- 【盈能】状态下回复3魔力：1+3=4魔力
- 用非强化攻击回蓝：
  - 射击：80%伤害，+3魔力（【盈能】）= 7魔力
  - 再等1回合：+3魔力 = 10魔力（可以装填）

### 2.2 最优循环

**循环1（打单发）**：
- 装填：6魔力 → 3弹药 + 3层【盈能】
- 强化爆射：625%
- 装填：6魔力 → 3弹药 + 3层【盈能】
- 强化射击：180%
- 等待2回合：+6魔力
- **5回合总输出**：625% + 180% = 805%
- **平均**：**161%/回合**

**循环2（爆射打空）**：
- 装填：6魔力 → 3弹药 + 3层【盈能】
- 强化爆射：625%
- 装填：6魔力 → 3弹药 + 3层【盈能】
- 强化爆射：625%
- 射击：80%
- 等待1回合
- **5回合总输出**：625% + 625% + 80% = 1330%
- **平均**：**266%/回合**

**最优循环**：循环2（爆射打空）
- **平均输出**：**266%/回合**

## 三、判官输出循环（修正）

### 3.1 翻转CD理解

**正确理解**：翻转有2回合CD，意思是：
- 回合1：使用翻转
- 回合2：CD中（不能用）
- 回合3：CD结束（可以用）
- 所以是**每隔1回合能用一次**

### 3.2 翻转循环（修正）

**循环**（3回合）：
- 回合1：阳击×2（+6平衡，达到+5上限，浪费1），伤害200%
- 回合2：翻转（450% × 1.25 = 562.5%），CD开始
- 回合3：等待或使用其他技能（200%），CD结束
- 回合4：可以再次翻转

**或者更优循环**（4回合）：
- 回合1：阳击（+3平衡），伤害100%
- 回合2：阳击（+3平衡，达到+5上限），伤害100%
- 回合3：翻转（562.5%），CD开始
- 回合4：等待（200%），CD结束
- **4回合总输出**：100% + 100% + 562.5% + 200% = 962.5%
- **平均**：**240.6%/回合**

**但需要考虑归中**：
- 回合1：阳击（+3平衡）
- 回合2：归中-2，阳击（+3平衡，当前+4平衡）
- 回合3：归中-2，翻转（当前+2平衡，翻转后-2平衡）
- 翻转伤害：200% + 50% × 2 = 300% × 1.1（+2平衡，+10%属性）= 330%

**更准确的循环**（考虑归中）：
- 回合1：阳击（+3平衡），伤害100%
- 回合2：归中-2，阳击（+3平衡，当前+4平衡），伤害100%
- 回合3：归中-2，翻转（当前+2平衡，翻转后-2平衡），伤害330%
- 回合4：等待（200%），CD结束
- **4回合总输出**：100% + 100% + 330% + 200% = 730%
- **平均**：**182.5%/回合**

**或者更优策略**（快速积攒）：
- 回合1：阳击×2（+6平衡，达到+5上限），伤害200%
- 回合2：翻转（562.5%），CD开始
- 回合3：等待（200%），CD结束
- **3回合总输出**：200% + 562.5% + 200% = 962.5%
- **平均**：**320.8%/回合**

**但需要考虑归中**：
- 回合1：阳击×2（+6平衡，达到+5上限），伤害200%
- 回合2：归中-2，翻转（当前+3平衡，翻转后-3平衡），伤害350% × 1.15（+3平衡，+15%属性）= 402.5%

- **3回合总输出**：200% + 402.5% + 200% = 802.5%
