# Spirit Weaver 平衡性分析（基于设计文档）

**分析依据**：设计文档 v2.0 中的数值设定  
**分析日期**：当前  
**注意**：此分析基于设计文档中的目标数值，而非代码实现现状

---

## 一、角色伤害输出对比（设计文档数值）

### 1.1 单次技能最高伤害

| 角色 | 基础技能 | 特技 | 能量技 | 最高爆发 |
|------|---------|------|--------|---------|
| **气宗** | 80% | 200% | 400% + 50%内伤 | **450%** |
| **剑圣** | 120% (2hit) | 100% (5hit) | 100% + 20%×10 = **300%** | **300%** |
| **魔导** | 80% / 150% | 120% (2hit) / 360% (3hit) | 装填 | **360%** (3hit) |
| **判官** | 100% | 100% | 200% + 50%×5 = **450%** | **450%** |

**初步结论**：
- ⚠️ **剑圣终结技伤害明显偏低**（300% vs 其他角色450%/360%）
- ✅ **气宗**和**判官**的爆发伤害最高（450%）
- ✅ **魔导**的强化连射伤害合理（360%）

## 二、资源效率与循环分析

### 2.1 气宗（Qi Master）

**资源系统**：
- 初始：10/10气
- 每回合回复：+2气
- 技能消耗：0/4/8气

**循环分析**：
- **轻击循环**：每回合80%伤害，无消耗
- **重击循环**：4回合一次重击（200%），平均50%/回合
- **崩山循环**：8回合一次崩山（450%），平均56.25%/回合

**评价**：✅ 循环合理，爆发伤害高

---

### 2.2 剑圣（Combo Master）

**资源系统**：
- 初始：0/10连击
- 每行动回复：+1连击
- 技能消耗：0/0/全部连击

**循环分析**：
- **疾风循环**：每回合120%伤害（2hit），无消耗
- **连斩循环**：每回合100%伤害（5hit），无消耗
- **终结技循环**：10次行动后使用终结技（300%），平均30%/回合

**⚠️ 问题**：
1. **终结技伤害偏低**：10连击才300%，资源效率仅30%/连击
2. **平均伤害输出偏低**：终结技循环平均30%/回合，远低于其他角色
3. **对比其他角色**：
   - 气宗崩山循环：56.25%/回合
   - 魔导强化连射：约72%/回合（考虑装填成本）
   - 判官翻转循环：约150%/回合

**建议调整**：
- 终结技伤害提升至：**150% + 25%×连击数**（10连击 = 400%）
- 或改为：**基础150%，每连击+1hit，每hit 50%伤害**（10连击 = 650%总伤害，但每hit独立计算防御）

---

### 2.3 魔导（Mana Master）

**资源系统**：
- 初始：10/10魔力，0弹药
- 每行动回复：+2魔力
- 装填：5魔力 → 5弹药
- 消耗弹药：回复2魔力

**循环分析**：
- **无弹药循环**：每回合80%-120%伤害
- **装填循环**：
  1. 装填：消耗5魔力 → 获得5弹药
  2. 强化连射：消耗3弹药 → 360%伤害 + 回复6魔力
  3. 净收益：-5魔力 + 6魔力 = +1魔力（净赚1魔力）

**评价**：
- ✅ 循环效率高，但可能过于高效
- ⚠️ **建议**：装填消耗提升至6魔力，或弹药生成降至4发，避免无限循环

---

### 2.4 判官（Balance Master）

**资源系统**：
- 初始：0（-5到+5范围）
- 每回合归中：-2点（向0靠拢）
- 技能偏移：±2点

**循环分析**：
- **积攒阶段**：3回合达到±5平衡值（阳击+2，阴击-2）
- **翻转爆发**：450%伤害（200% + 50%×5）
- **平均伤害**：450% ÷ 3回合 = **150%/回合**

**⚠️ 问题**：
- **归中速度过快**：每回合归中2点，5点平衡值3回合就归零
- **操作窗口期短**：玩家需要在3回合内完成积攒和翻转，否则平衡值会快速归零

**建议调整**：
- 归中速度改为：**1点/回合**，给玩家更多操作空间
- 或：翻转后不立即归零，而是缓慢归中

## 三、敌人难度分析

### 3.1 敌人属性（设计文档）

| 属性 | 数值 |
|------|------|
| HP | 100 |
| ATK | 8 |
| DEF | 3 |
| SPD | 80 |

### 3.2 战斗节奏分析

**玩家属性**：
- HP: 100, ATK: 10, DEF: 5, SPD: 300

**速度对比**：
- 玩家AV = 10000 / 300 = **33.3 tick**
- 敌人AV = 10000 / 80 = **125 tick**
- **玩家行动频率**：约 **3.75倍**于敌人

**伤害计算**：
- 玩家对敌人：10 ATK - 3 DEF = **7点/次**
- 敌人对玩家：8 ATK - 5 DEF = **3点/次**

**生存压力**：
- 玩家需要 **14次攻击**才能击败敌人（100 HP ÷ 7）
- 敌人需要 **33次攻击**才能击败玩家（100 HP ÷ 3）
- 由于玩家速度是敌人的3.75倍，玩家可以在敌人行动前行动多次

**⚠️ 问题**：
- **敌人难度明显偏低**：玩家生存压力小，战斗节奏偏慢
- **建议调整**：
  - 敌人速度提升至 **120-150**（玩家速度的2-2.5倍）
  - 或：敌人攻击力提升至 **10-12**（增加生存压力）
  - 或：敌人HP提升至 **150-200**（延长战斗时间）

## 四、魂灵系统平衡性

### 4.1 虚空领主（自律型）

**充能机制**（设计文档）：
- 触发间隔：**300 tick**
- 充能量：**10**
- 同步必杀伤害：**60**

**充能速度分析**：
- 每300 tick充能10% → 需要 **3000 tick**才能满灵压
- 假设1回合 ≈ 100 tick，需要 **30回合**才能满灵压
- 30%灵压需要 **9回合**

**⚠️ 问题**：
- **充能速度过慢**：30回合才能满灵压，节奏太慢
- **同步必杀伤害偏低**：60点伤害（约6倍基础攻击），低于其他魂灵

**建议调整**：
- 触发间隔缩短至 **200 tick**（20回合满灵压）
- 或：充能量提升至 **15**（20回合满灵压）
- 同步必杀伤害提升至 **80-90**（与其他魂灵持平）

---

### 4.2 幻影剑（协力型）

**充能机制**（设计文档）：
- 响应充能：**15-35**（根据伤害）
- 同步必杀伤害：**80**

**充能速度分析**：
- 每次响应充能15-35%
- 假设玩家每回合使用技能，**2-3回合**即可达到30%灵压
- **3-4回合**即可达到100%灵压

**评价**：✅ 充能速度快，适合高频攻击角色

---

### 4.3 以太商人（助理型）

**充能机制**（设计文档）：
- 资源消耗充能：**3-5/资源**
- 同步必杀伤害：**70**

**充能速度分析**：
- 每次消耗资源充能3-5%
- 假设玩家每回合消耗资源，**3回合**即可达到30%灵压
- **10回合**左右可达到100%灵压

**评价**：✅ 充能速度合理，适合资源管理角色

## 五、核心平衡性问题总结

### 🔴 严重问题（必须调整）

1. **剑圣终结技伤害偏低**
   - **设计文档数值**：100% + 20%×连击数（10连击 = 300%）
   - **问题**：伤害明显低于其他角色的能量技（450%/360%）
   - **建议**：提升至 **150% + 25%×连击数**（10连击 = 400%）
   - **或改为**：多hit机制，每连击+1hit，每hit 50%伤害

2. **剑圣平均伤害输出偏低**
   - **问题**：终结技循环平均30%/回合，远低于其他角色
   - **对比**：气宗56%，魔导72%，判官150%
   - **建议**：提升终结技伤害，或降低连击需求（8连击即可使用）

3. **虚空领主充能过慢**
   - **设计文档数值**：300 tick充能10%，30回合满灵压
   - **问题**：节奏太慢，玩家体验差
   - **建议**：触发间隔缩短至 **200 tick**，或充能量提升至 **15**

### 🟡 中等问题（建议调整）

4. **敌人难度偏低**
   - **设计文档数值**：SPD 80, ATK 8
   - **问题**：玩家速度是敌人的3.75倍，生存压力小
   - **建议**：敌人速度提升至 **120-150**，或攻击力提升至 **10-12**

5. **判官平衡值归中过快**
   - **设计文档数值**：每回合归中2点
   - **问题**：5点平衡值3回合就归零，操作窗口期短
   - **建议**：归中速度改为 **1点/回合**

6. **魔导装填循环效率过高**
   - **设计文档数值**：5魔力→5弹药，消耗3弹药回复6魔力
   - **问题**：净赚1魔力，可能形成无限循环
   - **建议**：装填消耗提升至 **6魔力**，或弹药生成降至 **4发**

### 🟢 轻微问题（可选调整）

7. **气宗内伤触发条件**
   - **设计文档数值**：耗气>6的技能才触发内伤
   - **问题**：只有崩山（8气）能触发，重击（4气）不能触发
   - **建议**：降低阈值至 **4气**，让重击也能触发内伤

8. **虚空领主同步必杀伤害**
   - **设计文档数值**：60点伤害
   - **问题**：低于其他魂灵（80/70）
   - **建议**：提升至 **80-90**，与其他魂灵持平
