# Spirit Weaver 平衡性分析报告

**分析日期**：基于当前设计文档 v2.0  
**分析维度**：伤害输出、资源回转、生存能力、操作复杂度、魂灵协同

---

## 一、角色伤害输出对比

### 1.1 单次技能伤害倍率（理论值）

| 角色 | 基础技能 | 特技 | 能量技 | 最高爆发 |
|------|---------|------|--------|---------|
| **气宗** | 80% | 200% | 400% + 50%内伤 | **450%** |
| **剑圣** | 120% (2hit) | 100% (5hit) | 100% + 20%×连击数 | **300%** (10连击) |
| **魔导** | 80% / 150% | 120% (2hit) / 360% (3hit) | 装填 | **360%** (3hit) |
| **判官** | 100% | 100% | 200% + 50%×|平衡| | **450%** (5平衡) |

**初步结论**：
- ✅ **气宗**和**判官**的爆发伤害最高（450%）
- ⚠️ **剑圣**的终结技伤害偏低（10连击才300%），可能需要调整
- ✅ **魔导**的强化连射伤害合理（360%）

---

### 1.2 资源效率分析（伤害/资源比）

#### 气宗（Qi）
- **轻击**：80% / 0气 = **∞**（无消耗）
- **重击**：200% / 4气 = **50%/气**
- **崩山**：450% / 8气 = **56.25%/气**

**结论**：崩山的资源效率略高于重击，符合设计预期。

#### 剑圣（Combo）
- **疾风**：120% / 0连击 = **∞**（无消耗）
- **连斩**：100% / 0连击 = **∞**（无消耗）
- **终结技**（10连击）：300% / 10连击 = **30%/连击**

**⚠️ 问题**：终结技的资源效率偏低。如果玩家积攒10连击，期望的伤害应该更高。

**建议**：
- 终结技基础伤害提升至 **150%**，每连击+25%，满连击时达到 **400%**
- 或者：终结技改为多hit，每连击+1hit，每hit 50%伤害

#### 魔导（Mana + Ammo）
- **无弹药射击**：80% / 0弹药 = **∞**
- **强化射击**：150% / 1弹药 = **150%/弹药**（同时回复2魔力）
- **强化连射**：360% / 3弹药 = **120%/弹药**

**结论**：弹药效率合理，但需要先装填（消耗5魔力）。

#### 判官（Balance）
- **阳击/阴击**：100% / 0资源 = **∞**
- **翻转**（5平衡）：450% / 0资源 = **∞**（但需要先积攒平衡值）

**结论**：判官的伤害完全依赖平衡值积累，需要多回合准备。

## 二、资源回转速度对比

### 2.1 资源获取速度

| 角色 | 初始资源 | 每回合获取 | 获取方式 | 满资源回合数 |
|------|---------|-----------|---------|-------------|
| **气宗** | 10/10 | +2 | 被动回复 | 0回合（已满） |
| **剑圣** | 0/10 | +1/行动 | 行动积攒 | 10回合（10次行动） |
| **魔导** | 10/10魔力，0弹药 | +2魔力/行动 | 被动回复 | 0回合（魔力已满） |
| **判官** | 0 | ±2/技能 | 技能偏移 | 3回合（达到±5） |

**分析**：
- ✅ **气宗**开局即可使用高伤害技能
- ⚠️ **剑圣**需要10次行动才能满连击，节奏较慢
- ✅ **魔导**魔力回复快，但需要装填才能使用强化技能
- ✅ **判官**3回合即可达到最大平衡值

### 2.2 资源消耗循环

**气宗循环**：
- 每回合+2气 → 4回合可用一次重击（200%），8回合可用一次崩山（450%）
- **平均每回合伤害**：约 **56.25%**（崩山循环）

**剑圣循环**：
- 每次行动+1连击 → 10次行动后使用终结技（300%）
- **平均每回合伤害**：约 **30%**（终结技循环）
- ⚠️ **问题**：平均伤害偏低，且终结技伤害也偏低

**魔导循环**：
- 装填（5魔力）→ 5发弹药 → 强化连射（360%）→ 消耗3弹药 → 回复6魔力
- **循环效率**：5魔力 → 360%伤害 + 6魔力（净赚1魔力）
- **平均每回合伤害**：约 **72%**（如果每回合都能行动）

**判官循环**：
- 3回合积攒平衡值 → 翻转（450%）
- **平均每回合伤害**：约 **150%**（翻转循环）

## 三、生存能力分析

### 3.1 基础属性对比

所有角色初始属性相同：
- **HP**：100
- **ATK**：10
- **DEF**：5
- **SPD**：300

**敌人属性**：
- **HP**：100
- **ATK**：8
- **DEF**：3
- **SPD**：80

**战斗节奏**：
- 玩家速度 300 → AV = 33.3 tick
- 敌人速度 80 → AV = 125 tick
- **玩家行动频率**：约 **3.75倍**于敌人

**生存压力**：
- 敌人每回合造成 8 点伤害（玩家DEF 5，实际伤害 3）
- 玩家需要 **33回合**才会被击败（100 HP / 3 伤害）
- 玩家可以在敌人行动前行动 **3-4次**

**结论**：当前敌人难度较低，玩家生存压力不大。

### 3.2 被动技能对生存的影响

- **气宗**：无生存加成
- **剑圣**：速度叠加（最高+50速度）→ 行动频率提升
- **魔导**：无生存加成
- **判官**：平衡正值时全属性+10%/点 → 最高+50%属性（包括HP、DEF）

**结论**：判官在平衡正值时生存能力最强。

## 六、核心平衡性问题总结

### 🔴 严重问题

1. **剑圣终结技伤害偏低**
   - 当前：100% + 20%×连击数（10连击 = 300%）
   - 建议：150% + 25%×连击数（10连击 = 400%）
   - 或改为：多hit机制，每连击+1hit，每hit 50%伤害

2. **剑圣平均伤害输出偏低**
   - 需要10次行动才能使用终结技，平均每回合伤害约30%
   - 建议：提升终结技伤害，或降低连击需求（8连击即可使用）

3. **虚空领主充能过慢**
   - 300 tick才能充能10%，30回合才能满灵压
   - 建议：缩短触发间隔至 **200 tick**，或提升充能量至 **15**

### 🟡 中等问题

4. **敌人难度偏低**
   - 玩家速度是敌人的3.75倍，生存压力小
   - 建议：提升敌人速度至 **120-150**，或提升敌人攻击力至 **10-12**

5. **判官平衡值归中速度**
   - 每回合归中2点，5点平衡值3回合就归零
   - 建议：归中速度改为 **1点/回合**，给玩家更多操作空间

6. **魔导装填成本**
   - 装填消耗5魔力，但强化连射只消耗3弹药，循环效率高
   - 建议：装填消耗提升至 **6魔力**，或降低弹药生成至 **4发**

### 🟢 轻微问题

7. **气宗内伤触发条件**
   - 当前：耗气>6的技能才触发内伤
   - 只有崩山（8气）能触发，重击（4气）不能触发
   - 建议：降低阈值至 **4气**，让重击也能触发内伤

8. **魂灵支援技能未完全实现**
   - 虚空领主和幻影剑的支援技能都是占位
   - 建议：尽快实现具体效果
