# BGM 版本说明

- `qi_battle_loop.wav` / `combo_battle_loop.wav` / `mana_battle_loop.wav` / `balance_battle_loop.wav`

  - 视为 `V2`
- `qi_battle_loop_v3.wav` / `combo_battle_loop_v3.wav` / `mana_battle_loop_v3.wav` / `balance_battle_loop_v3.wav`
  - 新一轮按职业特征重写后的对比版本
  - 视为 `V3`
- `qi_battle_loop_v4.wav` / `combo_battle_loop_v4.wav` / `mana_battle_loop_v4.wav` / `balance_battle_loop_v4.wav`
  - 基于进一步听感反馈继续修正后的版本
  - 视为 `V4`
- `qi_battle_loop_v5.wav` / `combo_battle_loop_v5.wav` / `mana_battle_loop_v5.wav` / `balance_battle_loop_v5.wav`
  - 根据更细的听感问题继续修正后的版本
  - 视为 `V5`
- `qi_battle_loop_v6.wav` / `combo_battle_loop_v6.wav` / `mana_battle_loop_v6.wav` / `balance_battle_loop_v6.wav`
  - 针对失真、长音波动、剑圣重拍结构和判官旋律轮廓继续修正后的版本
  - 视为 `V6`
- `combo_battle_loop_v7.wav` / `balance_battle_loop_v7.wav`
  - 基于最新反馈仅对剑圣和判官继续修正后的版本
  - 视为 `V7`
  - `qi` / `mana` 当前最新仍为 `V6`
- `combo_battle_loop_v8.wav` / `balance_battle_loop_v8.wav`
  - 在保留 `V7` 对比文件的前提下，对剑圣和判官继续修正后的版本
  - 视为 `V8`
  - `qi` / `mana` 当前最新仍为 `V6`
- `qi_battle_loop_v9.wav` / `combo_battle_loop_v9.wav` / `mana_battle_loop_v9.wav` / `balance_battle_loop_v9.wav`
  - 基于 `preview_pack_v1` 开源采样做的首轮“替换音色版”战斗 BGM
  - 视为 `V9`
  - 保留旧版合成器资源不覆盖，仅把游戏默认播放切到这一轮试听版
- `qi_battle_loop_v10.wav` / `combo_battle_loop_v10.wav` / `mana_battle_loop_v10.wav` / `balance_battle_loop_v10.wav`
  - 基于 `preview_pack_v2` 中的 `GeneralUser-GS.sf2` 做的“正常 GM 音色版”战斗 BGM
  - 视为 `V10`
  - 保留 `V9` 作为怪采样对比版，并把游戏默认播放切到这一轮
- `qi_battle_loop_v11.wav` / `combo_battle_loop_v11.wav` / `mana_battle_loop_v11.wav` / `balance_battle_loop_v11.wav`
  - 基于 `V10` 的继续修正版，重点处理鼓过重、鼓失真感和主旋律被压住的问题
  - 视为 `V11`
  - 保留 `V10` 作为上一轮正常 GM 配器版对照，并把游戏默认播放切到这一轮

## V1

- 早期版本在进入版本化命名之前已被覆盖
- 当前仓库中未单独保留 `V1` 文件

## V2 特征

- 四首曲子的整体语言还比较接近
- 节奏型、旋律轮廓、和声语汇的分化不够大
- 更像同一类 JRPG 战斗曲模板的不同变体

## V3 特征

- `气宗`
  - 主旋律尽量回到大调/五声化思路
  - 去掉更强的紧张半音感
  - 后 8 小节增加了第二旋律层，强化蓄势后爆发的推进
- `剑圣`
  - 保留可接受的半音经过与更锐利的短句
  - 重心放在高速推进和连续斩击感
  - 后 8 小节加入更密的辅助旋律线
- `魔导`
  - 保留半音色彩与法阵/能量感
  - 依旧以分解和弦、脉冲、晶体点缀为主
  - 后 8 小节补了悬浮式第二旋律层
- `判官`
  - 亮面与暗面做更明确的切换
  - 尝试用偏大调与带半音的小调色彩交替体现阴阳两极
  - 后 8 小节增加对应明暗两套的补充旋律线

## V4 特征

- 共同调整
  - 后 8 小节的“第二层”不再主要用短句对位
  - 改为更长音、更偏和弦托底的支持层
  - 继续保留 16 小节以上长度，但进一步拉开长音和短音交错的节奏设计
- `气宗`
  - 进一步收束到更偏大调、五声化的主旋律
  - 避免过强的紧张半音色彩
- `剑圣`
  - 保留高速推进感，但把后半段奇怪的第二层改成更稳的和声托底
  - 尝试减少“全程同密度切分”的听感
- `魔导`
  - 主旋律保留原先相对可接受的方向
  - 重点修后 8 小节支持层，让它更像法阵/能量底盘而不是额外抢戏旋律
- `判官`
  - 改成每 4 小节切一次调式块
  - 前 4 小节走中式大调五声音阶倾向
  - 后 4 小节切到更暗的小调块
  - 第二轮更多依赖底部支持层来拉开变化，而不是额外加太跳的对旋

## V5 特征

- 共同调整
  - 后 8 小节的支持层整体加强，不再只做很弱的背景
  - 支持层更偏长音和弦托底，而不是额外的短句抢旋律
- `气宗`
  - 开头主音和更多主旋律线条改成更长的长音
  - 减少附点感，让旋律更有长线条和中式气口
  - 保留后 8 小节较合适的和弦支持层方向，并把存在感再抬高
- `剑圣`
  - 后 8 小节第二层进一步改成更明确的绵延和弦支持
  - 试图形成“主体轻快、背后持续托住”的关系
- `魔导`
  - 不再让节奏骨架长期停留在同一种附点语感
  - 重点改成每 2/4 小节切换不同节奏模板
  - 后 8 小节支持层明显增强
- `判官`
  - 保留每 4 小节切换一次调式块
  - 把暗面旋律改成与亮面更接近的轮廓，而不是做反方向的生硬对比
  - 后半段支持层同样增强

## V6 特征

- 共同调整
  - 底鼓失真感整体减轻
  - 后半段支持层继续加强，但仍保持“托底”而不是抢主旋律
- `气宗`
  - 长音的音高波动明显减弱
  - 开头与主旋律进一步拉成长线条
  - 保留你认可的后 8 小节和弦支持层方向，并再强化一点存在感
- `魔导`
  - 保留你认为已经比较对味的诡秘旋律方向
  - 节奏模板继续维持按 2/4 小节切换的思路
  - 主要延续前一版，只做支持层与底鼓修正
- `剑圣`
  - 从“全程高速”改成更明确的“前半积攒、后半重击”
  - 每 4 小节内加入更清楚的重点重拍与重音落点
  - 后半段支持层再增强，尽量做出轻快主旋律 + 绵延托底的关系
- `判官`
  - 不再强调生硬的大小调对撞
  - 更改为偏“上行循环”与“下行循环”的轮廓对照
  - 节奏加入更多长音与不完全对称的句法

## V7 特征

- 仅更新 `剑圣` 与 `判官`
- `剑圣`
  - 不再做“一整小节只剩第一个音重”的难受重拍
  - 改为更接近“前 6 拍持续积攒，7 拍留空，8 拍重击”的 2 小节句法
  - 重击不再只有一下，而是改成更明确的三连重击收束
  - 后半段支持层继续加强，目标是做出主旋律轻快、背后和弦绵延的关系
- `判官`
  - 放弃大小调对撞
  - 改成每 4 小节一个轮廓循环：
    - 1-2 小节：低到高到低
    - 3-4 小节：高到低到高
  - 后 8 小节继续保留更明显的支持层
- `气宗` / `魔导`
  - 不再继续出 `V7`
  - 当前建议继续以 `V6` 为最新对比版本

## V8 特征

- 仅更新 `剑圣` 与 `判官`
- `剑圣`
  - 去掉上一版末端的三连重击写法
  - 改回更正常的高速旋律推进，并保留“前半积攒、后半留空后重落”的两小节句法
  - 重拍从三连改成更直接的单次重落，尽量减少听感上的别扭感
  - 后半段支持层再略微加强
- `判官`
  - 去掉半音化、暗面化的旋律处理，整体回到更稳定的大调听感
  - 继续保留每 4 小节一个轮廓循环：
    - 1-2 小节：低到高到低
    - 3-4 小节：高到低到高
  - 后 8 小节支持层仍保留，但和声材料也统一回到大调内
- `气宗` / `魔导`
  - 当前最新仍为 `V6`

## V9 特征

- 共同调整
  - 不再只是继续改旋律与节奏，而是首次把一部分主音色替换成开源采样
  - 本轮主要目的是让你直接听“配器/音色方向”而不是继续抽象试听单个采样
  - 仍然保留部分合成低频和节奏骨架，避免第一次采样化就把整体打散
- `气宗`
  - 主旋律改用 `ocarina`
  - 主鼓点改用 `frame drum`
  - 重要段落增加 `gong` 落点
- `剑圣`
  - 主旋律改用更锐利的 `strumstick`
  - 重拍骨架改用 `frame drum`
  - 后半段支持层改成较轻的 `folk harp`
- `魔导`
  - 主旋律与分解层改用 `clavisynth`
  - 后半段支持层改用 `hand chimes`
  - 保留部分合成低频，避免失去法阵/能量底盘
- `判官`
  - 主旋律改用 `folk harp`
  - 后半段支持层改用 `hand chimes`
  - 节奏骨架加入更克制的 `frame drum`，并补 `gong` 仪式感落点

## V10 特征

- 共同调整
  - 放弃 `V9` 那批偏异域、偏怪的主音色选择，不再拿零散采样硬拼主旋律
  - 改为使用 `GeneralUser-GS.sf2` 的标准 GM 乐器，通过 `py-meltysynth` 离线渲染主旋律、支持层和法阵/分解层
  - 尽量保留此前你已经基本认可的旋律与节奏骨架，本轮重点只修“配器正常化”
- `气宗`
  - 主旋律改为更稳妥的 `Flute`
  - 长音托底改成 `Strings Ensemble`
- `剑圣`
  - 主旋律改为更直接的 `Violin`
  - 后半段绵延支持层改成 `Strings Ensemble`
- `魔导`
  - 主旋律改为 `Polysynth`
  - 分解层改为 `Crystal`
  - 底盘与后半段支持层改成 `Warm Pad`
- `判官`
  - 主旋律改为更平稳的 `Clarinet`
  - 长音支持层改成 `Strings Ensemble`

## V11 特征

- 共同调整
  - 继续保留 `GeneralUser-GS.sf2` 的正常 GM 配器路线，不再回到怪采样
  - 全局下调鼓组源头电平，减轻底鼓/桶鼓的失真感与毛刺感
  - 进一步压低军鼓与帽子的存在感，避免鼓把主旋律盖过去
  - 总输出余量略微回收，减少整体挤压感
- `气宗`
  - 主旋律仅轻微前推
  - 鼓组改为更克制的托底，不抢长线条主旋律
- `剑圣`
  - 主旋律从 `Violin` 改为更靠前、更有攻击性的 `Overdrive Guitar`
  - 同时削弱后半段支持层与鼓组，让高速旋律重新站到前景
- `魔导`
  - 主旋律从偏糊的 `Polysynth` 改为更有辨识度的 `Saw Lead`
  - 分解层与支持层后撤，避免法阵层把主旋律吞掉
- `判官`
  - 主旋律从 `Clarinet` 改为更尖一点的 `Oboe`
  - 鼓和支持层后撤，让线条更容易被听见

## 备注

- 当前游戏代码默认播放的是 `V11`
- `V9` 仍保留，仅作为“首轮怪采样替换版”的对照
- 未带版本后缀的旧默认文件仍然保留，可随时切回
- 如果确认 `V6` / `V8` / `V10` / `V11` 中某一版更合适，再继续细修或固定为正式版本
